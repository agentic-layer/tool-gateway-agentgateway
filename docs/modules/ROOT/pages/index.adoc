= Agentgateway based Tool Gateway

This is an agentgateway based implementation of the Agentic Layer Tool Gateway.

The Tool Gateway Agentgateway Operator is a specialized Kubernetes operator that provides an agentgateway-based Tool gateway implementation for the Agent Runtime Operator ecosystem.
It manages `ToolGateway` resources and automatically generates agentgateway configurations to expose Tool workloads through a unified gateway interface.
